Focused Self Collective - Static site bundle

Files included:
- index.html (homepage)
- about.html, services.html, approaches.html, supervision.html, fees.html, contact.html, privacy.html, blog.html
- README.md (this file)
- supervision_agreement.pdf (sample supervision agreement ready to download)
- accessibility_statement.pdf (sample accessibility statement)
- mockup.png (simple visual mockup of the homepage)
- integrations.txt (instructions & code snippets for wiring forms/Calendly/Netlify/Formspree)

How to use:
1. Unzip the bundle.
2. Upload to any static host (Netlify, Vercel, GitHub Pages). If using Netlify and wanting form handling, leave the 'data-netlify' attribute in the contact form and add a Netlify _redirects or netlify.toml if needed.
3. Replace placeholders (e.g. Formspree ID) with your account values and add real images.
